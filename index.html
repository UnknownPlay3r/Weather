Weather Overlay for OBS

Setup Instructions

  1. Open OBS Studio
  2. Add a new "Browser" source to your scene
  3. Copy this URL for the overlay:
    weather-overlay.html
  4. Set these properties:
    • Width: 500
    • Height: 200
  5. Recommended settings:
    • ✓ Refresh browser when scene becomes active
    • ✓ Shutdown source when not visible
    • ✓ Enable browser source hardware acceleration

Features

  • Local time with AM/PM
  • Current temperature in Celsius
  • Weather conditions with color-coded icons
  • Location detection
  • Transparent background for easy integration

Troubleshooting

  • If location isn't showing, allow location access in your browser
  • If weather isn't updating, check your internet connection
  • For best results, keep the browser source active
